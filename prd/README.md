# Omad Arena PRD

Эта папка содержит продуктовую и техническую спецификацию backend-части Omad Arena.

## Документы
- `backend_prd.md` — основной PRD по backend: доменная модель, бизнес-правила, API-контракты, архитектура, этапы разработки.

## Как используем
1. Фиксируем изменения требований сначала здесь.
2. После согласования переводим задачи в реализацию backend.
3. На каждом этапе сверяем код с этим PRD.
